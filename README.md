# shelby-blob-metadata

> Fetch and display blob metadata stored on Shelby Protocol.

![Python](https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=python&logoColor=white)
![Shelby](https://img.shields.io/badge/Shelby-Protocol-c6ff00?style=flat-square&labelColor=0a0a0f)

## Usage
```bash
pip install -r requirements.txt

python main.py 0xYourAddress
python main.py 0xYourAddress --blob path/to/file.pdf
```
